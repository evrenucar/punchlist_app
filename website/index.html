Why this exists

My tasks lived in five places: a notes app, chat messages to myself, two abandoned to-do apps, & whatever paper was on the desk. Every tool I tried wanted an account, a sync service, or a subscription before it would let me type a first task. Then it wanted me to stay, so getting my own lists back out came down to whatever its exporter felt like giving me.

So I set one rule for this build: the whole app has to fit in a single file you can open from disk. No database, no server, no framework, no npm install. If it can't survive being emailed to yourself as an attachment, it's too heavy. Today's build is 185,841 bytes of native HTML, CSS & JavaScript.

Your data never leaves the browser. Tasks persist in local storage, export losslessly to JSON, & copy out as indented Markdown that pastes clean into Obsidian, Notion, or any text editor. There's no telemetry to switch off because the file doesn't contain any.

How it was built, commit by commit

I ran this as a spec-first build with Claude Code writing the implementation against my requirements & my live feedback. Every feature landed the same way: write the failing test, make it pass, rebuild the single file, then click through it in a real browser at 1440×900 and 390×844 before committing. The static suite sits at 58 tests across 1,067 lines. The app itself is 3,271 lines of JavaScript & 1,187 lines of CSS.

What the file does

✓Nested outline. Enter splits a task at the caret, Tab indents, Alt+arrows move items, & undo holds your last 40 board actions.
✓Markdown clipboard. Ctrl+C turns any selection into clean indented Markdown. Collapsed groups copy in full, children included.
✓Linked copies & shortcuts. Put one task in two places; edit either copy & both update. A shortcut jumps to the original instead.
✓Completed & Trash. Finished tasks hide on your schedule: immediately, after 7 days, or never. Deletes go to a restorable Trash by default.
✓Day timeline. Optional view from 06:00 to midnight with a current-time line & drag rescheduling snapped to 15 minutes.
✓Focus mode. One task full-screen with a running timer. Time accumulates per task & reads out against your plan, like "+10m vs plan".
✓Works on your phone. Drawer sidebar, 44 px touch targets, long-press drag. Light & dark themes with per-group color tints in both.
✓Your data, portable. Lossless JSON export & import. Off by default means invisible: disabled features add zero buttons to the screen.
