Help

Basics. Enter adds a task, Tab and Shift+Tab indent or outdent (a whole multi-selection at once), arrows browse, Alt+arrows move an item anywhere: through subtrees and across groups. Ctrl+Up/Down collapses or expands, Ctrl+Enter marks the selected task done, Ctrl+Alt+F opens focus mode (works on groups too).

Board shortcuts. Alt+A adds a group. Ctrl+Shift+Down expands everything; Ctrl+Shift+Up collapses everything. ArrowLeft jumps to the sidebar, ArrowUp from the top task reaches search.

Clipboard. Ctrl+C copies selected tasks as clean Markdown. Pasting inside the board places a linked copy by default; change this in Settings. Ctrl+X then paste moves tasks. Pasting Markdown from outside creates new tasks.

Linked copies. A linked copy is one task shown in two places: edit or complete either one and both update. A shortcut is a small pointer that jumps to the original instead.

Your data. Everything stays in this browser. Use Export JSON for a lossless backup and Import JSON to load it elsewhere.

Reminders. Alerts and browser notifications only fire while this page is open.

Touch. Press and hold a task or timeline block, then drag it.
